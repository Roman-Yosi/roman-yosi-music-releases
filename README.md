# Roman Yosi Music Releases

Official music release database for Roman Yosi.

## About

Roman Yosi is the stage name of Rohman, an independent musician from Lebak, Banten, Indonesia. This repository contains a public catalog of official music releases, release metadata, discography information, and artist-related data.

## Features

- Official discography
- Release metadata
- ISRC and UPC information
- Artist profile
- Music catalog database
- GitHub Pages website
